# Vinculum

Vinculum is a Kubernetes-native software delivery platform built around durable resources, agent drones, repository automation, review flows, and Helm-packaged runtime components.

This repository now uses a markdown memory bank as the main project context. The goal is to keep project knowledge split into smaller linked files so humans and LLMs can navigate it without treating one giant README as the only source of truth.
