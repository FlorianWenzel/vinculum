# Glossary

Back to index: [`README.md`](README.md)

## Terms

- `Vinculum` - the overall platform and central coordination layer
- `Drone` - a containerized agent identity with execution configuration
- `Requirement` - a durable user-facing request or product need, backed by markdown
- `Task` - derived technical work that can be assigned and executed
- `Review` - explicit review decision state around task output
- `Hive UI` - current browser control surface
- `Vinculum Code` - Forgejo instance branded for this project
- `Memory Bank` - the linked markdown context set under `docs/memory-bank/`

## Terminology caveat

Some project language is intentionally thematic. In implementation-oriented docs, prefer the more literal terms where possible.
